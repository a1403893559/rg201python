什么是HTML？

HTML是用来描述网页的一种语言。

  • HTML 指的是超文本标记语言 (Hyper Text Markup Language)
  • HTML 不是一种编程语言，而是一种标记语言 (markup language)
  • 标记语言是一套标记标签 (markup tag)
  • HTML 使用标记标签来描述网页

HTML标签

HTML 标记标签通常被称为HTML标签（HTML tag）

  1. HTML 标记标签通常被称为HTML标签
  2. HTML 标签是由尖括号包围的关键字
  3. HTML 标签通常是成对出现
  4. 标签是对中的一个标签开始，第二个标签是结束标签
